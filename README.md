# 💳 Credit Card Fraud Detection

A machine learning project that detects fraudulent credit card transactions using supervised learning techniques.

---

## 📌 Project Overview
Credit card fraud is a major financial problem. This project aims to build a predictive model that can distinguish between **legitimate** and **fraudulent** transactions based on transaction data.

The model is trained on historical transaction data and evaluated using standard machine learning metrics.
